# Web Application with Login System

Web application featuring user authentication, database persistence,
and dynamic pages using JavaScript.

## Features
- User login and registration
- Protected pages after authentication
- Database integration
- Dynamic content with JavaScript
- Linked navigation between pages

## Tech Stack
- HTML
- CSS
- JavaScript
- Database (MySQL )

## Functionality
- Main landing page
- Login and registration system
- Pages accessible only to authenticated users
- Persistent user data stored in database
